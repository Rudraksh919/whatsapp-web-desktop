# WhatsApp Web Desktop (Linux)

Unofficial WhatsApp Web wrapper for Linux using Electron.

## Features

- Tray icon support
- Desktop notifications
- Downloads saved in `~/Downloads/Whatsapp`
- Runs as a standalone app

## Installation

### Using `.deb` (Ubuntu/Debian)

# WhatsApp Web Desktop (Linux)

WhatsApp Web wrapper for Linux using Electron.

## Features

- Tray icon support
- Desktop notifications
- Downloads saved in `~/Downloads/Whatsapp`
- Runs as a standalone app

## Installation

### Using `.deb` (Ubuntu/Debian)

Download the latest release `.deb` (e.g., `whatsapp-web-desktop_1.0.4_amd64.deb` for x64, or the `i386`/`arm64` equivalents) from [Releases](https://github.com/Rudraksh919/whatsapp-linux/releases) and install:

sudo dpkg -i ./whatsapp-web-desktop_1.0.4_amd64.deb
